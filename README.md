# Finfra Homebrew Tap

Homebrew formulae for Finfra macOS helper daemons.

## Install

```bash
brew install finfra/tap/fwarrange-cli
```

## Formulae

| Formula | Description |
| :------ | :---------- |
| `fwarrange-cli` | Window layout management daemon for [fWarrange](https://github.com/Finfra/fWarrange_public) |
